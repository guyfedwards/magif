# magif

save gifs in your menu bar, click to copy

<img src="screenshot.png" width="400">

###installation
```
git clone git@github.com:guyfedwards/magif.git
cd magif
npm install
```

###run dev
```
npm start
```

###package electron app
```
npm run build
cp -R magif-darwin-64/magif.app /Applications
```
